This is a Motion Canvas TypeScript presentation project.

Use:
- npm run serve for preview
- npm run build/render if available
- TypeScript scenes under src/scenes
- Keep animations readable and presentation-friendly
- Prefer generator-based Motion Canvas patterns
- Ask before adding new dependencies
- Use colours present in the Everforest colourscheme (https://everforest.vercel.app/)
- Use a light theme for the slides
- Use British English for all text
- Images are located in the /assets/ directory
- All slides a required to have the Unilever logo and the University of Liverpool in the bottom right hand corner
